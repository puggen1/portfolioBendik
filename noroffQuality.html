About the project

A group project divided into two parts. one design and user testing part, and one implementing part.

The main goal was to create a page for Noroff (the school im studying at) that showed all major events of the year, example: school start, student council election etc.

This meant we also had to user test on the diffrent target audience.

The requirements

Requirements was as following:

  • user testing to get relevant feedback and see if the product works as intended.
  • store and access the relevant data on an api.
  • be able to filter the data so only relevant data is being showed.

The final version

The final product is a very simple and easy web page to use, there is not to many choices. the filtering of diffrent data is very fast, and it allows for secondary filters after the main filteris chosen.
